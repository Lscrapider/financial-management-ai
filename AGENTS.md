# Agent Instructions

每次新建对话时，先阅读本文件，并按以下要求选择和使用 skill / tool。

## 写代码/改代码

- 使用 Superpowers 插件及其适用 skill：做新功能、改行为、实现功能、修 bug、调试、计划、头脑风暴、并行 agent 协作、代码审查和收尾验证时，按 Superpowers 的流程执行。
- 使用 `scrapier-guidelines`：写代码时遵循实现逻辑、最小改动、验证标准、Spring Boot 分层规范。
- 使用 `codegraph_*`：分析“这个功能怎么流转”“这个方法谁调用”“改这里影响哪里”等代码结构和调用关系问题。

## 做前端设计

- 使用 `impeccable`：处理 UI 设计、重构、视觉打磨、交互、响应式、可访问性、信息架构。

## 语言要求

- 项目文档和代码注释使用中文。
